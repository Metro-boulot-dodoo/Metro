# Metro
## Installer et lancer le projet

### Lancer le backend
- Ouvrir 1 terminal dans le dossier Metro
- Copier coller les commandes suivantes dans le terminal   
Sur un système UNIX :
```shell
cd ./backend
cp .env.example .env
npm install
npm run dev
```

Sur un système Windows :
```shell
cd ./backend
copy .env.example .env
npm install
npm run dev
```

### Lancer le frontend
- Ouvrir 1 terminal dans le dossier Metro
- Copier coller les commandes suivantes dans le terminal   
Sur un système UNIX :
```shell
cd ./frontend
npm install
npm run dev
```

Sur un système Windows :
```shell
cd ./frontend
npm install
npm run dev
```

## Voir le projet
- Ouvrir un navigateur internet moderne (Firefox, Chrome,...)
- Se déplacer vers l'URL : ``http://localhost:5173/``
